== Description ==
This free blog site template, built on Bootstrap 4, features a sleek left sidebar and offers extensive customization options through the use of CSS variables (custom properties).
The template is designed to be fully responsive, ensuring a seamless user experience across various devices. With its modern and stylish design, 
this template is an ideal choice for bloggers looking to create an engaging and visually appealing online presence. 
Additionally, it includes features that make it easy to navigate and interact with the content, 
enhancing the overall user experience.

5.Customize everything in your theme including colors, typography, spacing, layouts.


== Installation ==

1- Log in to your WordPress dashboard, navigate to the Themes menu, and click “Add New.”
2- In the search field type "astr-blog" then click "Search Themes.".
3- Click "Install now". 
4- Activate the Themes.
